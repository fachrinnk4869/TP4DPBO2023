##TP4DPBO2023
Download Kode PHP pada link berikut ini [TP MVC](https://drive.google.com/file/d/1nBMAyhAOgDJvvMYqLhJ9qZRo14nN9XLM/view?usp=share_link).
1. Buatlah database berdasarkan kode tersebut
